迭代器与生成器

发表于 2016-03-17

迭代器

  • 迭代器接口： next()方法，调用该方法遍历所有值后，迭代器将引发异常StopIteration，让python停止遍历。
  • 创建自定义迭代器时，需要实现方法__iter__和next()

生成器

  • 同样使用next()方法
  • 使用yield而不是return

生成器表达式

  • 它们将立刻返回，因为它们根本没有创建任何结果——仅当试图遍历时，才会创建结果。

函数式编程

map和filter

  • map接收一个函数和一个可迭代对象（如列表或生成器），并返回一个列表，其中包含对可迭代对象的每个元素执行函数得到的结果。
  • filter也接受一个函数和可迭代对象，但只返回对可迭代对象执行函数后结果为true的元素。
